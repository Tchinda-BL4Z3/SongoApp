Placez ici le fichier audio MVett (format MP3) nommé `mvett.mp3`.

- Chemin attendu par le code : `assets/mvett.mp3`
- Android : il est recommandé d'ajouter `mvett.mp3` aussi dans `android/app/src/main/res/raw/` si le require échoue.
- Après avoir ajouté le fichier, rebuild the app.

Commands:

```bash
npm install
npx pod-install ios
npm run android # ou npm run ios
```
